HTML

Html je sve što vidimo na našem pretraživaču. Svaki tekst, dugme ili određeni deo web stranice je HTML.

CSS

Kada bi HTML bio ljudski skelet, CSS bi bili njegova koža, kosa i sve ostalo što čoveku daje, u našim očima, lep izgled.

JavaScript

U našem poređenju, Javascript bi bili mišići, koji pokreću naš skelet i čine ga dinamičnim.

PHP

Bez skriptnih serverskih jezika, kao sto je PHP, svi web sajtovi bi bili statične HTML stranice. Da nema njih, nikada ne bi mogli da napišete komentar na Facebook, a da isti bude sačuvan.

MySQL

Ti isti Facebook komentari, da bi bili sačuvani, potrebno je da se negde skladište, za to služe baze podataka i jezik za upravljanje SQL.

Frameworks

Oni predstavljaju biblioteke programskih jezika, koji nam olakšavaju rad pri kucanju i nude nam gotove komponente.

Moja filozofija web programiranja

Web programiranje i programiranje uopšte su kao i svaki drugi posao kojim želite da se bavite. Pre svega, potrebna je želja i volja za učenjem, vreme i požrtvovanost. Bez obzira što neki misle da je to naučna fantastika, učenje ovog “zanata” je dostupnije nego učenje, na primer, kako da budete profesionalni varioc. Sve što vam je potrebno su računar i internet konekcija, sve ostalo, što je potrebno za uspeh, je u vama samima. Ne treba biti uplašen, jer u suštini, svaka osoba je ista, a to znači da svako ima ono najpotrebnije, a to je MOZAK :).

by Mladen L.

Web Programiranje

Web programiranje obuhvata Frontend i Backend. Programiranjem postižemo to da statičku stranicu pretvorimo u funkcionalnu. Može se iskoristiti u raznim delovima jednog projekta.

Web i Grafički Dizajn

Dizajn je vizuelni vid komunikacije i dosta doprinosi korisničkom iskustvu. On pokriva korisnički interfejs i sve interesantno što možete videti na ekranu.

Baze Podataka

Sama reč nam kaže baze podataka nam služe za skladištenje informacija i podataka korisnika koji se preuzimaju sa servera određene web-stranice.

GitHub

Jedan od najbitnijih alata svakog progrmera jeste GitHub. Preko njega se verziraju kodovi i timovi developera rade dosta projekata upravo uz pomoć njega.

Command Prompt

Poznavanje rada na komandnoj liniji sa određenim komandama je svakako prednost. Značiće vam dosta prilikom korišćenja Linux OS i GitHub-a.

Responzivni dizajn

Veliki broj ljudi posećuje sajtove sa manjih uređaja pa je stoga reponzivni dizajn krucijalan. Danas sajt mora biti odgovarajući za sve uređaje.
